# Be The Hero

A simple project provide by **RocketSeat** Omnistack 11, a **web** and **mobile** aplication that contain login & register, pagination, and more.

## What's its used

**Back-end**
- Node
- Express
- Knex

**Front-end**
- React JS
- Axios


## How to run

**Back-end**

`yarn start`: to start

`yarn dev`: to start with watch

**Front-end**

`yarn start`: to start
